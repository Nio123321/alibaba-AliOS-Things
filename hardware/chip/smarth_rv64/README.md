# 概述
**smarth_rv64** 是基于平头哥C906核心smart SoC，主要用于架构寄的集成验证，调试仿真，以及FPGA应用评估的综合演示平台。

## 芯片介绍
关于C906芯片手册可参考平头哥官网 [C906芯片手册](https://www.t-head.cn/technology?icon=C906)

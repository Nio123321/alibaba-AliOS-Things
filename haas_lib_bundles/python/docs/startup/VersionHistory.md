# HaaS Python版本历史

&emsp;&emsp;
HaaS Python是阿里云IoT HaaS团队推出的HaaS物联网设备云端一体开发框架的重要组成部分。

&emsp;&emsp;
HaaS Python的目标是帮助中小开发者聚焦业务，实现设备安全上云，加速设备创新迭代，真正做到“Python也可以轻松开发智能设备”。

&emsp;&emsp;
本文主要介绍HaaS Python的固件版本信息。
